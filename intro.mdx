# O que é a 147.rest API?

Nós amamos dados, amamos programação e também adoramos proporcionar uma excelente experiência para desenvolvedores.

Nosso principal objetivo é criar a melhor API para desenvolvedores que precisam de informações de dados na internet, com todas as estatísticas e informações principais que você precisa.

# Como posso contribuir com a 147.rest API?
Fazendo uma doação ou apenas usando o serviço gratuito que a api fornece e lembre-se de divulgar para seus amigos.

# Chaves de API

Para obter sua Chave de API gratuita:
- Faça login com o discord em: https://147.rest/
- Veja e copie sua Chave de API na página do seu perfil: https://147.rest/perfil
