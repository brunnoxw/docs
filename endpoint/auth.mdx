# Introdução

- Nossa API é baseada na arquitetura REST com respostas no formato JSON.
- O número de requisições permitido atualmente: <b>até 1 requisição por segundo.</b>
- Os endpoints exigem uma chave de API única que inclui seu ID do discord e mais 32 caracteres por solicitação HTTP do usuário, conforme o exemplo abaixo:

```bash
'X-API_KEY': '687022032726392839-6990a81cda9d65f920235f5c53aee5f6'
```

- Você pode encontrar sua chave de API em: https://147.rest/perfil
